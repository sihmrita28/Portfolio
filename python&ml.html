Sentiment Analysis

Naan Mudhalvan Project

Home/ Sentiment Analysis

Objective

The objective of this project is to build a sentiment analysis model that can automatically classify product reviews into Positive, Neutral, or Negative categories. This allows businesses to quickly interpret customer feedback, monitor product performance, and improve customer satisfaction strategies.

Methodology

Data Transformation:
The dataset containing product reviews, ratings, and sentiments is first cleaned by removing missing or null entries. The review texts are then converted into numerical format using TF-IDF vectorization, enabling the model to process and learn from the textual data. Standard text preprocessing techniques like stop-word removal are applied.

Modeling and Metrics:
The data is split into training and testing sets using an 80-20 split. A Logistic Regression model is used for classification due to its simplicity and effectiveness in text classification tasks. Model performance is evaluated using key metrics like accuracy, precision, recall, and F1-score, and visualized with a confusion matrix.

Visualization:
A confusion matrix heatmap is used to display the number of correctly and incorrectly predicted sentiments. Additional plots such as bar charts can be created to show sentiment distribution and review lengths, if needed.

User Review Prediction:
A custom user input feature is added, where a user can enter a product review. The model processes the input and predicts the sentiment in real time using the same TF-IDF vectorizer and trained classifier.
